## FILMING TIPS

**Lighting:** Natural light from window or ring light
**Background:** Clean desk, maybe support software visible on second monitor
**Audio:** Use lavalier mic or quality headset mic
**Screen recording:** Use Loom, OBS, or built-in Mac screen recording

**Energy level:** Enthusiastic but professional
**Pacing:** Speak clearly, pause between major points
**Demo speed:** Slow enough to follow, fast enough to keep attention

**Post-production:**
- Speed up repetitive parts (file uploads, etc.) to 1.5x
- Add chapter markers in YouTube
- Include time stamps in description
- Add captions for accessibility
