## Quick Start (5 Minutes)

1. **Create a new Project in Claude**
   - Go to claude.ai
   - Click "Projects" → "New Project"
   - Name it "Customer Support Agent"

2. **Add Custom Instructions**
   - Copy everything from `custom-instructions.md`
   - Paste into Project's custom instructions field

3. **Upload Knowledge Base**
   - Customize the templates in `/knowledge-base/` with your company info
   - Upload all documents to the Project

4. **Test It**
   - Try the sample queries from `/demo-conversations/`
   - Verify escalation logic works correctly

5. **Integrate** (Optional)
   - Follow `setup-guide.md` for Slack/email integration
   - Or use Claude's API for custom integrations
